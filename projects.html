Train Time

A tool for displaying train schedules allows you to enter the first departure time and frequency and calculates the rest of the departure times for you. Originally this did not include the ability to edit or delete schedules which would be problematic in real life.


Trivia Game

This Trivia Game started as a one-trick pony. The questions and answers were hard-coded in the HTML making it impossible to change the questions without redoing much of the work.

I have moved the questions and answers into a JSON object that can be easily updated and used JQUERY to dynamically generate the layout.
